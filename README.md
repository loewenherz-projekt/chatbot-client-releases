# ChatBot Client — Downloads

Öffentliche Installer-Downloads für den **ChatBot Client** (Windows). Dieses Repository
enthält ausschließlich die fertigen Installer-Binärdateien — **kein Quellcode**.

## Download

**Neueste Version (immer aktuell):**

➡️ [`ChatBotClient-Setup.exe`](https://github.com/loewenherz-projekt/chatbot-client-releases/releases/latest/download/ChatBotClient-Setup.exe)

Alle Versionen unter [Releases](https://github.com/loewenherz-projekt/chatbot-client-releases/releases).

## Installation

Doppelklick auf `ChatBotClient-Setup.exe` — **keine Administratorrechte** nötig. Nach der
Installation startet der Client und öffnet die Oberfläche automatisch im Browser. Anmeldung
mit den Zugangsdaten von deinem Administrator (Server `https://cb.lhzp.de` ist vorausgewählt).

## Auto-Update

Der Client kann sich selbst aktualisieren (Button **„Nach Updates suchen"**). Neue Versionen
werden hier als Release veröffentlicht; der Client lädt sie automatisch von der „latest"-URL.
